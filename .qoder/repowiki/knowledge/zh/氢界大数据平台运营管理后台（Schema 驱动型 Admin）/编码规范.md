- 每个域目录必须包含 manifest.ts、schema.ts、repository.ts、types.ts 与 index.ts 五个文件，并通过 index.ts 统一导出供 bootstrap 发现
- 域清单 manifest.ts 通过 `order` 字段控制路由与菜单排序，`menuGroup` 决定所属分组，`routes[].hideInMenu` 控制是否出现在侧边栏
- 数据访问统一走 `useRepository(domain)` Hook，页面不直接依赖 Http/Mock 实现，切换数据源仅需修改 domains.config.ts
- Mock 数据由各域 repository.ts 在模块加载阶段调用 `registerMockData(domainName, data)` 注册，禁止在组件内硬编码数据
